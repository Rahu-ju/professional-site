Learning Path Tracker

Educational Goal-Setting App

Live Demo

This web app helps self-learners to track their progress and stay committed to their goals.

The Challenge

Self-taught learners often lose motivation because they can't see their cumulative progress. They need a way to track their journey and celebrate small wins.

My Solution

I built a goal-tracking web application where users can:

  • Set learning goals with clear titles
  • Log daily progress with detailed notes
  • View their complete learning journey from start to finish
  • Stay motivated by seeing how far they've come
Technical Implementation
  • Django backend with SQLite database
  • User authentication and personalized dashboards
  • Clean, intuitive interface built with Bootstrap
Personal Note

I created this tool while teaching myself Django. I was my own first user, and it genuinely helped me stay motivated during the challenging early days of learning. Seeing my progress accumulate day by day kept me going. Now others use it to track their own learning journeys.
